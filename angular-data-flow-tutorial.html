Data Flow in the Same Component

This is for sharing data between the view (HTML file) and the component (TS file)

Data Flow in Parent/Child Components

This is for sharing data between parent components and child components; in other words, "child" components that are nested inside "parent" components.

Data Flow in Separate Components

This is for sharing data between components that do not have a parent/child relationship; in other words, one component is not nested within another.
